# Physics Practical Learning Platform

A complete professional educational website for teaching secondary school Physics practical work and projects. Built with modern full-stack technologies.

**Author:** Group 4, 2026

## Features
- Interactive theory modules (Mechanics, Electricity & Magnetism, Waves)
- Step-by-step practical experiments
- Student projects gallery
- Timed multiple-choice quiz
- JWT authentication with student/teacher roles
- Student and teacher dashboards with statistics
- Responsive design (desktop, tablet, mobile)
- Professional UI with deep blue and cyan theme

## Tech Stack
- **Frontend:** HTML5, CSS3, JavaScript, Bootstrap 5, AOS animations, Font Awesome, Chart.js
- **Backend:** Node.js, Express.js
- **Database:** MongoDB (Mongoose)
- **Auth:** JSON Web Tokens (bcryptjs)
